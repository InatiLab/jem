jem.filters.dog_rotational_invariants(f, order=0)[source]¶

Rotational invariants of gaussian derivative operator

returns a list

Principal invariants and Frobenius norm See https://en.wikipedia.org/wiki/Invariants_of_tensors

Gaussian (Order 0):

2 rotational invariant: G0, |G0|

Gradient (Order 1):

1 rotational invariant: |G1|

Hessian (Order 2):

in 2D, 3 rotational invariants: Tr(G2), Det(G2), |G2| in 3D, 4 rotational invariants: Tr(G2), Det(G2), I2(G2), |G2|

jem.filters.gaussian(data, scale=1)[source]¶

Rotationally symmetric Gaussian filter in the fourier domain

Parametrized by a scale, resolution reduced by 2^(scale+1). Assumes the pixels are all the same size. Pad and crop to prevent wrap around junk.
